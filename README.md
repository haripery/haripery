# Hi, I'm Hari

**Product-focused Frontend Architect** with 10+ years of experience shipping 0-to-1 products at **Shopify** and **AWS**.

I specialize in bridging complex backend telemetry — like anomaly detection and IoT data — into intuitive, high-performance UI using Server-Driven UI (SDUI) and scalable design systems. Currently exploring AI-native workflows and agentic UI patterns.

## What I'm Doing

- **Senior Frontend Engineer at [Shopify](https://shopify.com)** — building high-integrity event tracking, Dynamic Creative Optimization, and pioneering "Contextual UI" with Model Context Protocols (MCP)
- **Writing** at [Near the Singularity](https://nearthesingularity.com) — my blog on frontend architecture, AI, and building software near the singularity
- **Building** open-source tools and side projects in the AI + frontend space
